## Expense-Tracker
A sleek, mobile-friendly Progressive Web App (PWA) to track expenses during trips.   Built with  HTML, CSS, JavaScript

This app helps to manage and monitor expenses across different categories (columns).  
It works **offline**, syncs data locally using **LocalStorage**, and provides a **summary dashboard** with export options.

---

## 🚀 Features

### 🧮 Expense Management
- Add expenses with name + amount
- Delete individual items
- Column-based categories (Travel, Shopping, Food, etc.)
- Rename categories instantly (inline editable)
- Add unlimited new columns
- Per-column total calculation
- **Grand Total** across all categories

### 📊 Summary Dashboard
- Pie chart visualization (powered by Chart.js)
- Category totals
- Grand total
- Export data:
  - JSON
  - CSV
  - Excel (.xlsx)

### 💸 Budget Tracking
- Enter budget limit (₹)
- Automatic “over budget” detection and warning

### 📱 PWA (Installable App)
- Install on Android / iOS / Desktop
- Offline-ready (service worker)
- App icon included
- Opens in standalone full screen mode

### 🎨 UI / UX
- Modern dark-themed UI
- Responsive layout
- Smooth horizontal scrolling columns
- Mobile-first design

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Chart.js** (for summary charts)
- **XLSX.js** (Excel export)
- **LocalStorage** (persistent offline data)
- **Service Worker** (PWA functionality)
- **Netlify** (deployment hosting)
